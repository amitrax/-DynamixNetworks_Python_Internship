# AI-Powered Chatbot with Sentiment Analysis

## Project Overview
This project is an AI-powered chatbot that interacts with users and detects the emotional tone of their messages (Positive, Negative, or Neutral).  
Based on the detected sentiment, the chatbot adapts its responses to make interactions more human-like and empathetic.

This project is developed as **Task-1** for the **Python Programming Internship**.

---

## Key Features
- Chatbot interaction using Streamlit
- Sentiment analysis using NLTK (VADER)
- Classification of messages into Positive, Negative, and Neutral
- Adaptive chatbot responses based on sentiment
- Context awareness using session memory
- Live dashboard for interaction logging and sentiment visualization
- No paid APIs used (fully free and offline)

---

## Technologies Used
- Python 3
- Streamlit
- NLTK (VADER Sentiment Analyzer)
- Pandas
